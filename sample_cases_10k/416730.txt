<p id="p_1">		This
application is filed for amending the prayer clause in Criminal
Miscellaneous Application No.9194 of 2007. 
</p>
 

 


 

<p id="p_2">		Learned
advocate Mr.Kanani for the applicant submits that though the order
passed by Fast Track Court was annexed, by mistake, specific prayer
was not made to quash and set aside the same. Hence, he filed this
amendment application for amending the prayer clause accordingly.
</p> 

 


 

<p id="p_3">		Learned
advocate Mr.Goswami for the respondents submits that when he raised
the said objection, this application for amendment is filed by
learned advocate for the applicant which should not be granted. 
</p>
 

 


 

<p id="p_4">		It
is an admitted fact that the order passed by the Fast Track Court was
annexed with the main application and if this application for
amending the prayer clause is allowed, no prejudice will be caused to
the respondents. 
</p>
 

 


 

<p id="p_5">		In
view of the above, this application is allowed. Amendment be carried
out forthwith. 
</p>

M.D.Shah, J )
